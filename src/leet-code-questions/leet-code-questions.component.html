Solution: This is a 2-pointer problem with the left pointer starting at the first element (0) and the right pointer starting at the last element (height.length - 1) of the array. As long as left is less than right we compare the elements at left and right. If the left element is less than the right element we set the area = height[left] * (right - left) and move the left pointer forward, otherwise we set the area = height[right] * (right - left) and move the right pointer backward. The area is only updated if the new value is greater than the current value.
